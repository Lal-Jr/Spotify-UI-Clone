# Installation

To use this project, follow the commands below:

```bash
# Clone this repository
git clone https://github.com/Lal-Jr/Spotify-UI-Clone.git

# Go into the repository
cd Spotify-UI-Clone

# Install dependencies
npm install 

# Run the project in local host
npm run dev
```
